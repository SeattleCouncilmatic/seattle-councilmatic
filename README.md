# Seattle Councilmatic

Seattle Councilmatic is a web application that enables Seattle residents to easily track Seattle City Council legislation, committee meetings, and council member votes. It’s built on the [Councilmatic](https://www.councilmatic.org/) platform.

## Features

- Find your Seattle City Council representative
- View the latest city council activity
- Track legislation (bills, resolutions, ordinances)
- See upcoming council meetings and agendas
- Search and filter legislation
- View council member voting records
